# Проект Музыка
## Описание
В рамках проекта проведено исследование поведения пользователей Яндекс.Музыки в Москве и Санкт-Петербурге по дням недели. Использована библиотека Pandas Python.
В ходе исследования проведена предварительная обработка данных: работа с пропущенными значениями, с дубликатами, проверено три гипотезы.
## Цель
Выяснить активность пользователей двух городов по дням недели, а также их жанровые предпочтения.
## Вывод
Имеются различия в активности пользователей Яндекс.Музыки по дням недели, но различия в жанровых предпочтениях двух городов несущественны.
